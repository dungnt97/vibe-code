## Why

Cần một trang blog chuyên nghiệp để tổng hợp tin tức và kiến thức về crypto. Hiện tại chưa có nền tảng nào trong project để chia sẻ nội dung — việc xây dựng blog giúp tạo kênh thông tin tập trung, dễ cập nhật và mở rộng.

## What Changes

- Thêm trang blog hoàn chỉnh với giao diện chuyên nghiệp, responsive
- Trang chủ hiển thị danh sách bài viết với thumbnail, tiêu đề, tóm tắt
- Trang chi tiết bài viết với markdown rendering
- Hệ thống phân loại bài viết theo category (News, Knowledge, Analysis)
- Trang About giới thiệu blog
- SEO-friendly với meta tags, Open Graph
- Dark mode mặc định phù hợp với aesthetic crypto

## Capabilities

### New Capabilities
- `blog-layout`: Layout chính của blog — header, navigation, footer, responsive design
- `article-system`: Hệ thống bài viết — markdown content, listing, detail page, categories
- `crypto-theme`: Theme chuyên nghiệp cho crypto — dark mode, typography, color scheme
